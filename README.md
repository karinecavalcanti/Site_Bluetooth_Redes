# 🔵 Tecnologia Bluetooth - Redes de Computadores

Este projeto é uma apresentação web interativa e dinâmica sobre a tecnologia **Bluetooth**, desenvolvida como trabalho teórico para a disciplina de **Redes de Computadores (EEL878)** da **Universidade Federal do Rio de Janeiro (UFRJ)**.

## Sobre o Projeto
O objetivo deste site é expor de forma visualmente atraente e academicamente rigorosa os conceitos teóricos do Bluetooth no contexto de redes sem fio. O conteúdo espelha o documento de pesquisa original da equipe, abordando desde a história da tecnologia até detalhes técnicos de arquitetura e segurança.

Para ilustrar o tema de redes, a interface utiliza um efeito visual interativo de nós conectados (*nodes* e arestas) ao fundo, simulando a formação de topologias como **Piconet** e **Scatternet**.

## Tecnologias Utilizadas
* **HTML5**: Estruturação semântica e divisão em múltiplas páginas.
* **CSS3**: Estilização com tema *dark mode*, efeito *glassmorphism* (vidro fosco) nos cartões e detalhes em azul neon.
* **JavaScript**: Lógica de integração e animação.
* **tsParticles**: Biblioteca leve utilizada para o renderizar o grafo de rede dinâmico no plano de fundo.

## Estrutura do Projeto
O conteúdo está dividido de forma modular para facilitar a leitura:

* `index.html`: Menu de navegação principal, introdução histórica e referências bibliográficas.
* `conceitos.html`: Definições, método FHSS e topologias de rede.
* `frequencia.html`: Espectro de 2,4 GHz, taxas de transmissão e evolução (Clássico, BLE, 5.x).
* `arquitetura.html`: Pilha de protocolos, relação com o Modelo OSI, perfis e aplicações práticas (IoT).
* `seguranca.html`: Mecanismos de proteção (criptografia, pareamento) e vulnerabilidades.
* `conclusao.html`: Considerações finais sobre o impacto da tecnologia.
* `style.css`: Regras globais de estilo e responsividade.
* `script.js`: Configurações da animação vetorial do fundo.

## Como Executar Localmente
O projeto é totalmente estático e não necessita de banco de dados ou execução de servidores no *backend*.

1. Faça o download ou clone este repositório.
2. Certifique-se de que todos os arquivos `.html`, `.css`, `.js` e as respectivas **imagens** estão reunidos na mesma pasta raiz.
3. Dê um duplo clique no arquivo `index.html` para abri-lo em qualquer navegador moderno (Chrome, Firefox, Edge, Safari).
